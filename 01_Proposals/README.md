# Proposals

Add one file per proposal. Suggested naming: YYYY_ClientName_ShortTitle.md

Suggested structure per proposal:
- **Client:** Name, sector, size
- **Brief:** What they asked for
- **Minkowski read:** How we framed the challenge
- **Approach:** What we proposed
- **Team:** Who was involved
- **Format:** Duration, venue, structure
- **Investment:** Budget range (can be approximate)
- **Outcome:** Won / Lost / Pending — and any learnings
