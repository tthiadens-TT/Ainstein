# Pricing

Add rate cards, pricing structures, and worked examples here.

Suggested structure per pricing file:
- **Type:** Day rate / Project fee / Programme package / Modular pricing
- **Scope:** What is included
- **Rate:** Amount or range
- **Margin notes:** Any cost assumptions or margin targets
- **Exceptions:** Known deviations or client-specific adjustments
- **Valid from:** Date range for this rate
